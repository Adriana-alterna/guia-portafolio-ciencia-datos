¿Qué es GitHub y por qué importa?

Git es un sistema de control de versiones que te permite llevar registro de tus proyectos de forma profesional.

GitHub es una plataforma en línea donde puedes compartir esos proyectos, colaborar con otros y construir tu portafolio.

Git y Github a fondo

¿Qué debe tener tu portafolio?

  • 📝 Un archivo README.md claro y atractivo
  • 📊 Notebooks con visualizaciones limpias
  • 📁 Carpetas bien organizadas
  • 💡 Proyectos propios o basados en datasets públicos
